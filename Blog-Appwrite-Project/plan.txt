#Profile Page/Dashboard

1. User Info Section
Profile picture (avatar).
Full name / username.
Short bio / tagline.
Location, website, social links (optional).
2. Stats Overview
📝 Number of blog posts written.
❤️ Total likes/upvotes.
💬 Total comments received.
👥 Followers / Following (if you add social feature).
3. User Content
My Posts → List of all blogs written by the user.
With edit/delete option (if owner).

✏️ Edit Profile → Update name, bio, avatar.
⚙️ Account Settings → Change email, password.
🚪 Logout button.

📑 Pages Required for Intermediate Blog App
1. Authentication Pages

🔑 Login Page → For existing users to sign in.

📝 Signup Page → For new users to register.

🔄 Forgot Password / Reset Page → Optional, but professional.

2. Main Content Pages

🏠 Home / Feed Page

Shows latest blog posts from all users.

Featured / trending blogs section.

🔍 Explore / Discover Page

Browse blogs by categories/tags.

Recommended posts / popular authors.

📖 Single Blog Page (Post Detail)

Displays full blog content.

Like, comment, and share options.

3. User-Generated Content Pages

✍️ Create Blog Page → Rich text editor (write + upload image).

✏️ Edit Blog Page → Modify existing blog.

🗂️ My Blogs Page → List of user’s own blogs (with edit/delete).

📌 Bookmarks / Saved Posts Page (optional).

4. Profile & Account Pages

👤 Profile Page (public view)

Shows user info, bio, and their blogs.

⚙️ Edit Profile Page (private)

Update name, bio, profile pic, social links.

🔧 Account Settings Page

Change password, email, preferences.

5. Social / Community Pages (Optional, but nice)

👥 Followers / Following Page → List of connections.

💬 Comments Management Page → User’s comments on blogs.

📨 Notifications Page → Alerts for likes, comments, follows.

6. Utility / Other Pages

❓ About Page → About the platform or team.

📬 Contact Page → Support or feedback form.

⚖️ Terms & Privacy Pages → Legal docs.

🚫 Error Pages (404 / 500) → For broken links or server errors.
